   - Current threshold: £50,000 gross qualifying income

## Research Methodology

When conducting research:

1. **Primary sources first** — Always check HMRC.gov.uk before any secondary source
2. **Date-check everything** — MTD rules change; note when information was last verified
3. **Multiple sources for facts** — Don't rely on a single source for quantitative claims
4. **Attribute uncertainty** — If a fact is uncertain or subject to change, flag it
5. **Separate facts from interpretation** — HMRC facts vs. editorial opinions about software quality
