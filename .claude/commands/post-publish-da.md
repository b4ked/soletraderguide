# Post Publish DA — SoleTraderGuide.co.uk

Use this command only after the post has been committed and pushed.

This is an advisory post-publication assessment. It does not block the publication that just went live.

## Purpose

- Evaluate the newly published post for authority-building value
- Capture follow-up improvements without reopening the publish gate
- Write one report file into `reports/`

## Inputs

- Published MDX file
- Post slug
- Final URL

## What To Review

- topic depth versus search intent
- internal linking strength
- trust and clarity for YMYL content
- commercial balance and disclosure
- likely topical-authority contribution
- obvious follow-up opportunities across guides, tools, reviews, or comparisons

## Report Format

Write:

`reports/da-report-[YYYY-MM-DD]-[slug].md`

Include:

1. Post summary
2. DA impact score out of 40
3. What strengthens authority already
4. What limits authority today
5. Priority follow-up actions

Keep the report actionable and concise. Avoid generic SEO filler.

## Output Block

```text
DA_REPORT: COMPLETE
FILE: reports/da-report-[YYYY-MM-DD]-[slug].md
SCORE: [n]/40
TOP_ACTIONS: [brief summary]
```
