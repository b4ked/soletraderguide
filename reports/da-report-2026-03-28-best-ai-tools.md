### Priority: HIGH — Add GOV.UK citations for tax claims

The post makes two specific tax claims that require authoritative citation on a YMYL-adjacent site:

1. "AI tools cannot replace HMRC-recognised accounting software for MTD obligations" — cite [https://www.gov.uk/guidance/use-making-tax-digital-for-income-tax](https://www.gov.uk/guidance/use-making-tax-digital-for-income-tax)
2. "AI subscription costs are allowable expenses if wholly and exclusively for business" — cite [https://www.gov.uk/expenses-if-youre-self-employed](https://www.gov.uk/expenses-if-youre-self-employed)

Without citations, these claims are asserted without evidence — a known E-E-A-T weakness flagged by Google's quality rater guidelines for financial content. Adding two GOV.UK inline links directly strengthens the post's authority signals.

**Proposed action:** Update `src/content/blog/best-ai-tools-uk-freelancers-2026.mdx` — add GOV.UK links inline at the relevant claims. Optionally add a `<SourceList>` component at the foot of the post.

**Route:** Write-Up Agent (content edit) → QA/Reviewer Agent.

---

### Priority: MEDIUM — Develop a link-earning angle to lift the post above commodity content

The current post is a competent AI tools roundup but will not earn backlinks organically because this format is published prolifically. To lift this post to "link-worthy", consider one of the following angles for a follow-up update:

**Option A — UK freelancer survey data.** Commission or reference a simple survey: "We asked 200 UK freelancers which AI tools they use for work." Even a small sample published as a data point gives journalists something citable that does not exist elsewhere.

**Option B — Expand the "not recommended" section.** A "3 AI tools UK freelancers should avoid paying for (and what to use instead)" angle is contrarian, shareable, and more likely to be cited by blogs and freelancer communities than a standard "best of" list.

**Option C — Add a cost calculator.** A simple interactive "AI tools budget calculator" (e.g. select tools → see monthly/annual cost in GBP) would make this post 2–3x more linkable and directly supports the site's tools strategy.

**Route:** Recommendation to Write-Up Agent (for content expansion) via SEO Agent review. This is a medium-term improvement — not a blocker, but should be scheduled before the post is used in any outreach.
